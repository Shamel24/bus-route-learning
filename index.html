Labeled Data

Supervised Learning

The model learns from labeled examples, mapping inputs to known outputs so it can predict labels for new, unseen data.

  • Requires labeled training data
  • Common tasks: classification, regression
  • Examples: spam detection, price prediction, image recognition
  • Algorithms: Linear Regression, Decision Trees, SVM, Neural Networks

Try it: drive the bus, and let it learn your route

Rename the store types if you like, pick one, and click along the streets to drop labeled store pins. Then drive the bus with the D-pad — every store it passes gets recorded, in order, as your demonstrated route. Once you've visited a few stops, train the model: it learns a rule connecting store type + location to visit order, so it can suggest a route for any set of stops, including ones you never drove to yourself.

Unlabeled Data

Unsupervised Learning

The model works with unlabeled data, finding hidden patterns or groupings on its own without predefined answers.

  • No labeled training data required
  • Common tasks: clustering, dimensionality reduction
  • Examples: customer segmentation, anomaly detection, topic modeling
  • Algorithms: K-Means, Hierarchical Clustering, PCA, Autoencoders
